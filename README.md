# O'zbekiston Poezlar Uchun Online Chipta Olish Tizimi

O'zbekiston bo'ylab poezlar uchun online chipta olish tizimi.

## 📋 Loyihaning Tavsifi

Bu loyiha o'zbekiston bo'ylab harakat qiladigan poezlar uchun online chipta olish tizimi. Foydalanuvchilar oson va tezkor tarzda chipta sotib olishlari, rout va vaqtni bilib olishlari mumkin.

## 🛠️ Texnologiyalar

- **HTML5** - Veb sahifa tuzilishi
- **CSS3** - Uslublanish
- **JavaScript** - Dinamik funksiyalar
- **Responsive Design** - Barcha qurilmalarga mos

## 🚀 Boshlash

### O'rnatish
```bash
git clone https://github.com/AkobirBarotov/online-tictet.git
cd online-tictet
```

### Ishga tushirish

Tuzilgan HTML fayllarni brauzerda oching yoki lokal server bilan:

```bash
# Python 3 bilan
python -m http.server 8000

# Yoki Node.js bilan
npx http-server
```

Keyin `http://localhost:8000` ga kirib sahifani oching.
